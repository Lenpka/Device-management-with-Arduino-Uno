# Device-management-with-Arduino-Uno.
Этот проект создан студентами 2го курса бакалавриата факультета наук о материалах МГУ: Константиновым Леонидом и Гумировым Ильгизом. Цель проекта - создать многоканальную систему управления исполнительными устройствами с помощью Python с поддержкой цифрового переключения частотного управления, и мониторинга состояния каналов через Serial-интерфейс.
